Research & Open Source Pipelines

Active computational projects and software tools built for astronomical analysis.

1. Galamo: Automated Morphological Classification

An open-source Python package for comprehensive galaxy analysis using DECals Legacy Imaging.

  • Integrates ML and statistical methods for morphology classification, kinematics, photometry, and spectral analysis.
  • Codebase: www.galamo.org

2. AGN Flux Disentanglement via PFVG Method (Active)

Studying AGN flux variability from the COSMOS Casata Survey.

  • Pipeline built to disentangle AGN flux contribution from total galaxy flux.
  • Authors: Madhu Sudan, Jashanpreet Singh Dingra, Ankit Sarkar & Hum Chand

3. Galspar & RAGA: Galactic Morphological Study (Active)

Automated pipelines for precise photometric extraction and morphological modeling.

  • Galspar: Performs exact measurements on standard FITS images.
  • RAGA: Automated pipeline for statistical morphology studies via GALFIT.

4. vdisp_fit: Stellar Velocity Dispersion Pipeline (2025)

A Python pipeline to measure stellar velocity dispersion in galaxies using SDSS data.

  • Method: Direct spectral fitting by convolving a MILES stellar template with a Gaussian broadening function utilizing robust χ2 minimization.
  • Authors: Jashanpreet Singh Dingra & Gulab Chand Dewangan (IUCAA)

5. Dynamics of Open Star Clusters (2025)

Implemented an N-body simulation to model gravitational interactions and long-term dynamical evolution.

  • Utilized fourth-order Runge-Kutta (RK4) method.
  • Analyzed stellar trajectories and cluster stability over cosmic timescales.

6. Crab Pulsar Spin Down Rate over 5 Years (2024)

Analyzed archival X-ray timing data (2015-2023) to measure its spin-down rate and braking index.

  • Confirmed rotational energy loss due to magnetic dipole radiation and investigated short-term timing noise and glitches.
